Bases de datos. Conceptos

Bases de datos: Llamamos bases de datos a un conjunto de datos relacionados entre sí que están centralizados y que tienen una mínima redundancia.

Sistema gestor de archivos: es un programa o conjunto de programas que maneja la base de datos.

Registro: es el conjunto de todos los datos referidos a una entidad almacenada en la base de datos. Todos los registros del mismo tipo se almacenan en el mismo deposito de información llamados ficheros.

Campo: es cada uno de los datos simples que componen un registro. Todos los registros de un mismo fichero deben tener los mismos campos.

Dominio: Conjunto de posibles valores que puede tomar un campo. Los dominios mas usuales son alfanumérico o texto, numérico, fecha y hora, etc.
